React Recipes
================

This application was originally created to learn more about React and how it can be integrated as a front-end within a Rails application.

Ruby on Rails
-------------
This application requires:

- Ruby 2.7.1
- Rails 6.0.3.1
- PostgreSQL
- Yarn for Assets & Dependencies
